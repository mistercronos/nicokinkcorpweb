Information About Dragon Escape

Dragon Escape is the vol number 2 of Dragon Survivor. The game starts in a prison. The user has been captured by the dragons and he has to escape with some of the captured kinkies.
It is up to the user to find out what will happen. Dragon Escape was created in winter 2017 and released in august 2017. It was coded mainly by Nicolas Kress, NicoKink's CTO.
